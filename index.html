RESEARCH

Deep Active Learning | Undergraduate Researcher
UC Berkeley

Active learning is a supervised learning framework in which rather than building a model that iterates through a labelled dataset, the model is given unlabelled data and chooses on its own which data points would be most valuable if labelled. Obtaining unlabelled data is easier than obtaining labelled data, and a strong active learning model would be more efficient at learning if it were proficient in choosing optimal data points query for labels.
